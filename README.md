# Pratik Kudkikar — 3D Portfolio

A modern 3D portfolio website built with Vite, React, Three.js, Tailwind CSS, and Framer Motion.

## Tech Stack
- **Vite + React** — fast development and build
- **Three.js / @react-three/fiber** — 3D geometric hero scene
- **Tailwind CSS v3** — utility-first styling
- **Framer Motion** — smooth animations

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
